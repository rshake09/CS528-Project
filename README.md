# CS528-Project - Air Drawing Recognition System
CS528 Final Project

## Overview
Our project aims to build a wearable, finger-mounted stylus that allows users to "draw in the air" using hand motions. The system uses an IMU connected to an ESP32-S3 to capture accelerometer and gyroscope data, which is transmitted to a computer and visualized in real time. 

Our goal is to translate natural hand movements into a drawing interface with gesture-based controls.
